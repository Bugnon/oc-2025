Cours 0 – Introduction au module web

Une seule resource à connaître : ocinfweb.github.io

Devoirs à la maison (28.10.2025)

  • Créer un repositoire GitHub pour le projet
  • Créer un fichier README.md et y documenter dans les grandes lignes :
    • Le thème choisi
    • Le cahier des charges
    • La maquette du site web
  • Créer la structure du site web dans le repositoire (dossiers et fichiers HTML vides)
  • Envoyer le lien vers le répositoire à raphael.holzer@eduvaud.ch et denis.dreano@eduvaud.ch

Objectifs

  • Apprendre à créer un site web.
  • Modifier son apparence avec une feuille de style.
  • Utiliser canvas pour créer des animations.
  • Déployer un site web.

Planning

Le module aura lieu entre les vacances d'automne et de Noël. Un planning détaillé se trouve sur la page d'accueil de ce site.

Le projet

Vous allez créer un site web individuel. Le site doit avoir un certain nombre de caractéristiques qui doivent être documenté dans le cahier des charges qui doit être validé au début du cours et doit contenir une maquette du site.

Le projet sera évalué durant une présentation finale à la fin du module.

Les outils

Nous utiliserons les outils suivants durant le module :

  • Les macs du gymnase
  • VS Code ainsi que l'extension HTML Preview pour la programmation.
  • Google Chrome et Chrome DevTools pour explorer et débugger des sites web.
  • GitHub, GitHub Pages et GitHub Desktop pour héberger le code, le partager et déployer le site web.

    • Structure d'un site web

    Un site web minimal se compose de fichiers HTML qui contiennent le contenu du site. Voici un exemple minimal de document HTML : 
                
                |/
                |_ www/
                |     |_cours/
                |     |      |_cours1.html
                |     |      |_cours2.html
                |     |_tp/
                |     |   |_ tp1.html
                |     |   |_ tp2_html
                |     |
                |     |_ organisation.html
                |    
                |
                |_ css/
                |     |_style.css
                |
                |_ js/
                |    |_animation.js
                |    |_fonctions.js
                |
                |_ images/
                |        |_ image1.png
                |        |_ image2.jpg
                |
                |_ index.html
